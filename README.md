# EIMA JAX-RS REST API


## Running the application locally

First build with:

    $mvn clean install

Then run it with:

OS X/Linux:

    $ java -cp target/classes:target/dependency/* com.EIMA.Main
Windows:

    > java -cp "target\classes;target\dependency\*" "com.EIMA.Main"
